## Commandes utilisés
- `npm create vite@latest`
- `npm install`
- `npm i react-router` // Pour la barre de navigation
- `npm install gh-pages --save-dev` // Pour le déploiement

## Commandes utilisables
### Pour tester en local
- `npm run dev`
### Pour publier le site
- `npm run predeploy`
- `npm run deploy`
